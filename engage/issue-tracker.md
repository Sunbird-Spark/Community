# Issue Tracker

File issues and track progress on the Sunbird Spark issue tracker.

* Jira (primary tracker): [project-sunbird.atlassian.net →](https://project-sunbird.atlassian.net/)
* GitHub Issues (portal): [SunbirdEd-portal Issues →](https://github.com/Sunbird-Spark/sunbird-spark-portal/issues)
* GitHub Issues (mobile): [SunbirdEd-mobile-app Issues →](https://github.com/Sunbird-Spark/sunbird-spark-mobile-app/issues)
* GitHub Issues (knowlg-service): [knowledge-platform Issues →](https://github.com/project-sunbird/knowledge-platform/issues)
* GitHub Issues (lern-service): [lern-service Issues →](https://github.com/project-sunbird/lern-service/issues)

When filing an issue, please include:

* Spark version (from the portal footer or kubectl get deployment -n default)
* Steps to reproduce
* Expected behaviour
* Actual behaviour
* Browser/device (for frontend issues)
* Relevant logs (for backend issues)
