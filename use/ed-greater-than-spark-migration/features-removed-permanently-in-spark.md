# Features removed permanently in Spark

| Feature                                       | Replacement                                        |
| --------------------------------------------- | -------------------------------------------------- |
| Print Service (question paper PDF generation) | Removed — no replacement                           |
| Old Report Service                            | Replaced by built-in native reports on the portal  |
| Textbook deletion                             | Removed — textbooks can be retired but not deleted |
| Auto Batch Creation (post-publish)            | Removed — batches must be created explicitly       |
| Shallow Copy Publishing                       | Removed — content must be independently published  |
| Legacy (non-QuML) assessment approach         | Removed — QuML spec only                           |
| Obsrv v1.x helmcharts                         | Replaced by Obsrv add-on (new helmcharts)          |
