# Targeted programmes

{% hint style="info" %}
**Add-on required:** Manage Learn is an optional Spark add-on. It is disabled by default.
{% endhint %}

Manage Learn provides structured programme management capabilities beyond standard course enrolment. It is designed for scenarios where learners need to complete real-world tasks, not just consume digital content.

#### Projects

A project is a set of defined tasks that a learner must complete in the real world. Each task has a description, evidence requirements such as photos or text, and a deadline.&#x20;

Common use cases include:

* School improvement plans, where head teachers document infrastructure improvements
* On-the-job training milestones, where trainees submit evidence of workplace tasks
* Community health worker activity tracking

#### Observations

An observation is a structured data-collection exercise. An observer, such as a school inspector or programme manager, completes a QuML-based question set while visiting a facility or reviewing a learner's work. Observations generate structured data that can be aggregated for programme-level insight.

#### Surveys

Surveys are one-time or recurring questionnaires sent to targeted groups of learners or administrators. Survey data is aggregated and reported at the programme level.
