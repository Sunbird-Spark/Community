# Rich and diverse content

Spark supports a wide range of content formats, so learners get the right experience for each content type.

| Format             | Description                                         | Player                          |
| ------------------ | --------------------------------------------------- | ------------------------------- |
| PDF                | Document content                                    | PDF Player (V2 web component)   |
| Video              | `MP4`, `WebM` — adaptive streaming optional         | Video Player (V2 web component) |
| H5P                | Interactive HTML5 content                           | H5P Player (V2 web component)   |
| EPUB               | E-book format                                       | EPUB Player (V2 web component)  |
| Interactive (ECML) | Rich interactive content built with the ECML editor | ECML Player (V2 web component)  |
| Audio              | `MP3` and other audio formats                       | Audio Player (V2 web component) |

All players are V2 web components. They can be embedded in any web application, not just the Spark portal. This lets adopters build custom learner-facing applications while reusing Spark's content players.

{% hint style="info" %}
**For developers:** `ECML` is the format name used in developer documentation and APIs. In the learner-facing portal UI, ECML content is shown as **Interactive**.
{% endhint %}
