# Lex

**Organisation:** Infosys\
**Domain:** Enterprise workforce development and L\&D\
**Scale:** Global deployment for Infosys's workforce

Lex is a cloud-first, mobile-first enterprise L\&D platform built on the Sunbird building blocks. It demonstrates that the Sunbird architecture — designed for public-sector education — translates equally well to corporate learning contexts. Lex is accessible anytime, on any device, and supports both self-directed and programme-led learning.
