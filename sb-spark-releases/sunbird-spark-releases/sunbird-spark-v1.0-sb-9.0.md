# Sunbird Spark v1.0 (SB 9.0)

Release notes will be published here on the Spark v1.0 release date.

The Spark v1.0 release will include:

* Full Kubernetes-native deployment via the one-click installer
* Merged service architecture (knowlg-service + lern-service)
* YugabyteDB as the primary transactional database
* New React 19 portal and React native mobile app
* Built-in admin reports (no Obsrv required)
* Add-on architecture for optional features
* VAPT certification (in progress)
