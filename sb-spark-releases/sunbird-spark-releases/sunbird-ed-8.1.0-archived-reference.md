# Sunbird ED 8.1.0 — archived reference

Note: Sunbird ED 8.1.0 is the final release of the Sunbird ED series. Sunbird Spark (SB 9.0) is the successor. These release notes are retained as a reference for existing ED adopters.

Released: 16 December 2025

#### Key highlights

Backend framework modernisation:

* Migrated from Akka 2.5 to Apache Pekko 1.0.3 — removes Akka Business Source Licence dependency
* Play Framework upgraded from 2.4–2.8 to 3.0.5
* Scala upgraded from 2.11/2.12 to 2.13.12
* Node.js upgraded from v12 to v22.17.1 LTS for knowlg-mw-service and discussion-middleware

Frontend:

* Angular portal upgraded from v14 to React 19

Architecture:

* Learning Service officially deprecated — Assessment APIs moved to inQuiry Assessment Service
* Node.js runtime upgraded to v22 for middleware services
